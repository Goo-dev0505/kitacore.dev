01 — BASIC

基本設定・画像構成

土台になるパラメータ群

まず入れとくべき設定。ここを理解してないと、どんな呪文を書いてもイメージと違うものが出てくる。

ASPECT RATIO
--ar / --aspect
アスペクト比。--ar 16:9 --ar 3:2 --ar 1:1。スマホ縦は9:16、横長動画は16:9、カメラ写真風は3:2が定番。
VERSION
--v
バージョン指定。--v 7 で最新V7を明示。省略するとデフォルトモデルが使われる。
NIJI
--niji
アニメ・マンガ特化のNiji Journeyモードへ切り替え。--niji 6 など。人物・キャラクター系に強い。
TILE
--tile
シームレスパターンを生成。テクスチャや壁紙、繰り返し柄が必要なときに。
PROFILE
--p / --profile
自分のMidjourneyプロファイル設定を反映させる。独自スタイルの保存と呼び出しに使う。
DRAFT
--draft
下書きモード。高速・低コストで構図確認。本番生成の前の下見に使う。
EXCLUDE
--no
除外指定。--no text, watermark でテキストや透かしを排除。後述の「ちょっとした技」も参照。
QUALITY ※V7非推奨
--q / --quality
品質設定。V7ではデフォルトが最高品質のため不要。V6以前との互換で残っている。
V8.1 --q 4 で一貫性が超強化される。
V8.1
--hd
ネイティブ2K解像度で生成。V8.1 デフォルトで適用される。高精細な出力が必要なすべてのシーンで有効。
02 — STYLE

スタイル・表現の調整

AIの"解釈"をどこまで許すか

数値一つで全体の雰囲気ガラッと変わる。写真寄りにしたいのかポップにしたいのかによって全然違う設定になる。

パラメータ範囲効果と使いどき目安値
--s / --stylize 0–1000 標準は100。上げるとAIが芸術的に暴走しだす。下げると素直な出力に近づく。 写真:50 / ポップ:700
--style raw / --raw ON/OFF AIの"勝手な補正・演出"を切る。余計な加工なしで写真リアル系向き。 写真系には必須
--w / --weird 0–3000 奇抜さのつまみ。変な絵を意図的に出したいときに。上げすぎると崩壊するので注意。 ポップ感:200
読むポイント --stylize と --weird は別軸。stylizeはAIの「芸術的解釈の強さ」、weirdは「非常識な方向への引っ張り具合」。両方上げると予測不能な面白い出力が出ることもある。
03 — RANDOM

ランダム性・再現性

同じ絵をもう一回出したいか、毎回違うものが欲しいか

用途で使い分けるパラメータ群。再現性が必要なプロジェクトと、バリエーション探索では設定が真逆になる。

パラメータ値効果使いどき
--seed 任意の整数 ランダム種を固定。同じseedなら同じ条件で近い絵が出る。 シリーズ制作・再現が必要なとき
--chaos / --c 0–100 構図・色のランダム性。0で安定、100で毎回バラバラな構図に。 探索:20 / ミニマル:0
--stop 10–100 生成を途中で止める。80で止めるとラフ感・スケッチ調になる。 描き込みすぎを防ぎたいとき
プロンプト重み :: 数値 要素の比重を変える。cat::2 dog::1 で猫要素を2倍に。 要素のバランス調整
--r / --repeat 2–40 同じプロンプトで複数回まとめて生成。バリエーションが一気に欲しいとき。 候補出し・比較
04 — REFERENCE

参照・一貫性

「この画風で」「この顔で」を画像URLで伝える

参照系パラメータはウェイト調整が肝。数値の意味を把握してないとイメージ通りに動かせへん。

パラメータウェイト用途コツ
--sref URL --sw 0–1000 画風・スタイルをURLで参照。「この雰囲気で描いて」を画像で伝える。 --sw 500が標準。低くすると薄い参照になる。
--sref random — ランダムなスタイルコードを当てる。宝くじ感覚で使う。 当たると神スタイルが出る。
--cref URL --cw 0–100 キャラクター顔参照。「この顔で描いて」を画像で伝える。 --cw 0にすると顔だけ似せる。全身は似せない。
--oref URL --ow 1–1000 万能な要素参照。顔以外の特徴も含めて反映させる。 srefとcrefの中間的な挙動。
--iw 0–3 画像プロンプト全体の影響度を調整する。V8.1 復活・精度向上。 --iw 2でほどよく強い影響に。
--sv 7 固定値 V8.1 スタイル参照バージョン7。従来比4倍の安定性。--sref使用時に必ず追加すると効果が安定する。 --sref との組み合わせ必須
顔だけ参照の最強コンボ

全身を似せたくないとき・顔中心のキャラ再現には --cref URL --cw 20 --iw 2 の組み合わせが最も効く。--cw 20で顔中心に絞り、--iw 2で参照強度を上げる。このセットは覚えといて損なし。

AI Unlock — 参照パラメータ
この言葉を知ると引き出せること
「--sref」「--cref」「--oref」の違いを理解したうえでAIに聞くと、ユースケース別の最適なウェイト設定を提案してもらえる。「参照ってどうやるの？」ではなく固有名で質問するのが精度を上げるコツ。
Prompt ExampleMidjourneyのV7で、--crefと--srefを同時に使うとき、顔の類似度を最優先にしつつ画風も参照したい場合のウェイト設定（--cw / --sw）の推奨値と、その理由を教えてください。
05 — MODE

処理モード

速さ・コスト・公開設定のフラグ

生成の優先度とプライバシーを制御する。用途によって使い分けると無駄なGPU消費を抑えられる。

STANDARD
--fast
高速モード。通常の優先度で生成。基本はこれ。
ECONOMY
--relax
低優先モード。待ち時間は伸びるがGPU消費を抑えられる。急いでない一括生成向き。
PREMIUM
--turbo
超高速モード。コスト2倍だが圧倒的に速い。締め切りがあるときに。
PRIVATE (Pro限定)
--stealth
非公開モード。ギャラリーや他ユーザーのフィードに出ない。商用・クライアント案件向き。
PUBLIC
--public
stealth解除。公開状態に戻す。
V8.1
Prompt Shortener
長すぎるプロンプトをMJが自動で短縮する機能。V8.1から搭載。/shorten コマンドで重要語彙だけ残した最適化プロンプトを提案してくれる。
06 — RECIPE

活用例レシピ

「なぜこれを入れるのか」まで書いたコピペレシピ

設定値だけ見ても意味がわからへんから、判断の根拠まで書いてる。そのまま末尾に貼って使えばOK。

📷 光多めの写真風
--ar 3:2 --raw --s 50 --hd
  • --rawでAI補正をオフにして写真に近づける
  • --s 50でアート性を抑えて自然な質感を残す
  • 3:2はカメラ的な定番比率
🎨 色強めのポップアート調
--s 700 --c 20 --w 200 --hd --sv 7
  • --s 700で大胆で派手な色使いを解放
  • --c 20で適度に構図を崩してポップ感を出す
  • --w 200で少し奇抜さを足して"ポップ感"を増す
🌫️ 背景だけ柔らかく
--s 300 --chaos 10 --hd
  • --s 300で柔らかい描写になる中間値
  • --chaos 10でちょい揺らぎのナチュラルなボケ感
  • 主題を壊さず背景の空気感だけ変えたいとき
👤 顔だけ参照を強める
--cref URL --cw 20 --iw 2 --sv 7
  • --cw 20で低めにして顔中心の反映に絞る
  • --iw 2で参照画像の影響をほどよく強くする
  • 全身を似せたくないときの最強の組み合わせ
🔲 シンプルなミニマル構図
--s 80 --c 0 --stop 80 --hd
  • --s 80で控えめな描写にして余白を生かす
  • --c 0で構図が乱れずミニマルな静けさが出る
  • --stop 80で描き込みすぎる前に止めてスッキリ感確保
📐 プロンプト構成テンプレ（迷子防止）
[Subject] + [Style] + [Mood] + [Material/Technique] + [Lighting/Color] + --hd --sv 7 --params
呪文は"鍵"、パラメータは"つまみ"。Style → Mood → パラメータの順で整理すると組み立てやすい。

08 — COMPOSITION

構図（Double系）

二つのイメージを使った特殊構図

演出の幅をグッと広げる構図技法。プロンプトに組み込むことで独特の視覚効果が出せる。

08-1
Double Exposure
二つのイメージを重ねて混ぜる構図。顔×風景、シルエット×夜景など。内側の世界と外側の世界を溶かすような見せ方が特徴。
Overlay型 — 透かして重ねる
08-2
Double Frame / Split Frame
一枚の中を2コマに分割して並べて見せる構図。左：全身 / 右：顔アップ、Before / Afterなど。比較したい・二つの視点を同時に見せたいときに使う。
Side-by-side型 — 横に並べる
08-3
Triple Frame / Multi Panel V8.1
3分割グリッド構図。V8.1のGrid Modeと相性抜群。ビフォーアフターや3段階変化、コンタクトシート風の見せ方ができる。
Grid型 — 3列・多面構成
08-4
Liminal Overlay V8.1
リミナルスペース（誰もいない不気味な中間空間）にダブルエクスポージャーを重ねた構図。現実と非現実が溶け合う独特のムードになる。
Liminal + Overlay型 — 中間世界
09 — TIPS

ちょっとした技

--no を使った「出したくないもの」の封印術

これを知ってると出力の制御が格段に上がる。特にNSFW・著作権キャラの混入を防ぐのに有効。

🔞
エッチなのを封印
--no nsfw, nude
MJは標準でブロックされるが、際どい肌色を減らしたいときに追加で入れると効果的。V8.1 --noの効きがさらに強化された。
⚔️
暴力を控えめに
--no blood, gore, violence
アクション映画レベルに抑えたいときに有効。V8.1 より精度が高い。
©️
版権キャラを排除
--no copyright, trademark
「ミッキーっぽさ」が勝手に混ざるのを防ぐ。キャラ名も入れると更に効果的。V8.1 NSFW・版権対策が最強レベルに強化された。
