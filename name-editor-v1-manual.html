Version Guide

どのバージョンを
選ぶ？

V1はベース。まずここから始めて、物足りなくなったら上位バージョンへ。

03
Quick Start

30秒で
始める

ダウンロードして開くだけ。インストール不要。

1
HTML ファイルを開く
note記事からダウンロードした KITAcore-name-editor.html をダブルクリック。ChromeやSafariで開く。
2
白いページを右クリック
白いキャンバス上で右クリックすると「左右に分割」「上下に分割」のメニューが出る。
→ 右クリック → 左右に分割
3
コマをクリックしてセリフを入れる
コマ（パネル）をシングルクリックするとテキスト入力モーダルが開く。セリフやト書きを入力して「保存」。
→ クリック → セリフ入力 → 保存
4
ダブルクリックで下絵を描く
コマをダブルクリックすると描画モーダルが開く。ペンで構図のラフを描き、AIプロンプトも入力できる。
→ ダブルクリック → 描く → 保存
5
PNG で書き出す
左サイドバーの「↓ PNG書き出し」ボタンでネーム全体を画像として保存。GeminiやFlowに貼り付けて使う。
→ PNG 書き出し → Geminiに貼付

05
Operation 01

コマを割る

右クリック一発でコマを分割できる。繰り返し操作で複雑な割りも作れる。

操作 方法 結果
左右に分割 右クリック → 左右に分割 コマが左右2つに分かれる
上下に分割 右クリック → 上下に分割 コマが上下2つに分かれる
サイズ調整 分割線をドラッグ コマのサイズ比率を自由に変更
コマの削除 右クリック → このコマを削除 そのコマを削除（兄弟コマが拡張）
メニューボタン コマをホバー → 右上の ⋯ ボタン 分割/削除メニューが開く（右クリックと同じ）
// コマ割りの基本パターン例
右クリック
→ 上下に分割　　// まず上下でページを区切る
上コマを右クリック
→ 左右に分割　// 上段を左右で分ける
分割線をドラッグ
→ サイズ調整　　// 強調コマを大きく
⚠ TIPS
  • ・分割は何回でも繰り返せる（4コマ・8コマ・それ以上も可）
  • ・コマを削除すると元には戻せない（V1にはUndoなし）→ V3ならUndo可
  • ・分割線はピクセル精度でドラッグ調整できる
  • ・斜めの分割はV3の機能。V1では横/縦のみ

Operation 02

セリフを入れる

コマをシングルクリックするとテキスト入力モーダルが開く。入力したセリフは縦書き・青文字でコマ内に表示される。

手順 操作 詳細
① モーダルを開く コマをシングルクリック 「01. テキスト入力」モーダルが開く
② セリフを入力 テキストエリアに入力 「セリフ」「ト書き」どちらでもOK。改行も使える
③ 保存 「保存」ボタン または Enter コマ右上に縦書き青文字で表示される
④ 修正 コマを再度シングルクリック 同じモーダルが開き、内容を編集できる
⑤ 表示を隠す サイドバー「セリフ・ト書き」チェックを外す 全コマのセリフ表示をOFF（PNG書き出しにも反映）

Operation 03

下絵・プロンプト

ダブルクリックで描画モーダルを開く。構図ラフとAIプロンプトを同じ場所で管理できる。

手順操作詳細
① モーダルを開く コマをダブルクリック 「02. 下絵・プロンプト」モーダルが開く
② ツール選択 ツールバーで選択 ペン / 消しゴム / 色変更 / 線幅変更 が選べる
③ ラフを描く キャンバス上でドラッグ 640×400pxのキャンバスに構図メモを描く
④ プロンプト入力 下部テキストエリア AI画像生成用のプロンプトを英語で入力
⑤ 保存 「保存」ボタン 下絵は50%透過でコマに表示。プロンプトは赤文字
WORKFLOW — AI漫画の使い方
1. ネームをV1で作る（コマ割り + セリフ + プロンプト）
2. PNG書き出し → ネーム画像を取得
3. キャラ画像と一緒にGemini or Flowに貼り付ける
4. 「赤文字のpromptを削除してカラー漫画を作って」と指示
5. AI生成画像を受け取る

Operation 04

保存・書き出し

4つの保存方法がある。目的に合わせて使い分ける。

PNG
PNG 書き出し
ネーム全体を1枚の画像として保存。SNS投稿・Gemini/Flowへの入力・共有に使う。
サイドバー → ↓ PNG 書き出し

サイドバー → シナリオを保存

JSON 保存
コマ構成・セリフ・プロンプト・下絵を含む全作業状態を保存。あとで再開できる。
サイドバー → 保存 (.json)
↑
JSON 読込
保存したJSONを読み込んで作業を再開。ページタイトルも復元される。
サイドバー → 読込 (.json)

トグル 効果 PNG書き出し 使いどころ

FAQ

よくある質問

Q コマを間違えて削除してしまった。元に戻せる？
V1にはUndo機能がない。コマ削除は取り消せない。こまめにJSONで保存しておくことを強くすすめる。作業前・コマ割り完了後・セリフ入力後の3タイミングで保存する習慣をつけると安心。Undoが必要な場合はV3を使う。
Q 複数ページのネームを作りたい
V1は1ページ固定。複数ページが必要ならV2を使う。V2はページ管理機能を持つ。または、V1でページごとにHTMLファイルを作ってJSONで管理するという運用も可能。
Q 斜めのコマ割りができない
V1は縦・横の分割のみ対応。斜めコマ割りはV3の機能。対角分割・複数コマ対応はV3で実装されている。
Q PNG書き出しの画質が荒い
html2canvasを使用しているため、ブラウザのズームレベルや画面解像度によって変わる場合がある。書き出し前にブラウザのズームを100%にすることと、Retinaディスプレイ環境では2倍解像度で出力される点を確認する。書き出したPNGをGeminiに貼り付ける際は元の解像度のまま使うのが精度面で有利。
Q AIに投げるプロンプトはどう書けばいい？
基本は英語で。「キャラ名・ポーズ・カメラ角度・スタイル・雰囲気」を短く書く。例：hero, dramatic close-up, manga style, intense expression, backlit。GeminiやFlowに投げるときは、別途キャラ参照画像と一緒に渡すと精度が上がる。
Q スマートフォンから使える？
HTMLファイルはスマホブラウザでも開けるが、右クリックメニューが使えないため操作性が落ちる。コマのホバーメニュー（⋯ボタン）から分割・削除は可能。基本的にはPC・Mac推奨。
Q V5はまだないの？
note記事 #334「ちょっとやらかした。昨日V1を出したのに、もうV5ができた話」で言及されているとおり、V5は存在する。V2+V3の統合版にあたる位置づけ。詳細は該当記事を確認。
